====================================================
SECTION 2: OPENENV CONCEPT
====================================================

OpenEnv is a standardized framework for building AI environments.

Core loop:
env.reset()
state = env.state()

while not done:
    action = agent(state)
    state, reward, done, info = env.step(action)

Core components:
- Observation: What agent sees
- Action: What agent can do
- Reward: Feedback for action
- Done: Task completion signal
- Info: Extra metadata
